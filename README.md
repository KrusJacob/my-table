📊 #Table App

🚀 ##Features

📋 Display data in a table with columns:
    * Name 
    * Date
    * Numeric value
    * Actions
    
➕ Add new records via modal form
✏️ Edit existing records
🗑️ Delete records with confirmation
🔍 Search across all table fields (with debounce)
↕️ Sorting for all columns (type-aware)
🎨 Clean UI using Ant Design components
⚡ Optimized rendering with hooks (useMemo, useCallback)
🔎 Global state management (Zustand)

🛠️ ##Tech Stack:
    * React
    * TypeScript
    * Ant Design
    * Zustand (state management)
    * Day.js (date handling)
    * Vite (build tool)

📦 ##Installation
```bash
git clone <your-repo-url>
cd <project-folder>
npm install
npm run dev
```
